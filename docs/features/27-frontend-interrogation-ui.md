# 27 - Frontend Interrogation UI

## Objectiu
Crear la UI d'interrogatori per mostrar preguntes i respostes.

## Abast
- Ruta `/game/interrogation`.
- Panell amb pregunta actual i resposta mock.
- Llista lateral d'historial mock.

## Requeriments
1. UI responsive.
2. Mostra pregunta i resposta principals.
3. Mostra 3 entrades d'historial.

## Criteris d'Aceptacio
- Renderitza títol "Interrogation".
- Renderitza pregunta principal.
- Renderitza 3 entrades d'historial.

## Pla de Tests (Integracio)
1. Renderitza títol i pregunta.
2. Renderitza 3 entrades d'historial.
